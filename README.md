# English-Learning-Tool
This project was developed as part of a Computer Programming course and focuses on creating an interactive English learning application for beginner-level learners using MATLAB. The aim of the project was to design a simple, user-friendly educational tool that supports vocabulary learning and basic language acquisition through interactive exercises

Project Overview

The goal of this project was to design an educational program that teaches children how to write and spell English words. The application integrates text, sound, and visual elements to enhance learning and improve engagement. The interface was designed to be intuitive and child-friendly, allowing users to interact easily through typed input and keyboard controls.

The program also provides feedback, tracks learning progress, and adapts to user performance by focusing on areas where the learner makes mistakes. This helps create a personalized learning experience and supports gradual skill improvement.

Features

	•	Text-based learning materials
	•	Sound feedback for responses
	•	Visual elements and images to support learning

Interactive Interface
	•	Intuitive and child-friendly design
	•	Keyboard input handling
	•	Support for special keys (e.g., delete, backspace)
	•	Smooth navigation between exercises

Adaptive Learning
	•	Identifies mistakes and weak areas
	•	Focuses practice on challenging words
	•	Provides personalized learning experience

Feedback System
	•	Immediate feedback for correct and incorrect responses
	•	Encouraging responses to keep learning engaging

Progress Tracking
	•	Stores user progress

Cross-Platform Compatibility

	•	All required files included in the project folder
